# oktsec Homebrew Tap

Official Homebrew tap for [oktsec](https://github.com/oktsec/oktsec), runtime security for AI agents.

## Install

```bash
brew install oktsec/tap/oktsec
```

## Upgrade

```bash
brew upgrade oktsec
```

## Uninstall

```bash
brew uninstall oktsec
```

## About

oktsec intercepts AI agent tool calls before execution. 217 detection rules, session management with AI analysis, tamper-evident audit trail. Works with Claude Code, Cursor, Windsurf, OpenClaw, and 17+ agent tools.
